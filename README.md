## Research Program Summary

The **Extended Phenotype Theory of Law (EPT)** applies Dawkins's (1982) extended phenotype theory to legal and institutional systems. Legal norms function as cultural replicators (memes) with fitness = P(transmission) × P(compliance) × P(enforcement). Institutions are the material extended phenotypes these replicators build for their own reproduction.

The program combines:
- **EPT** (Extended Phenotype Theory) — norms as replicators, institutions as extended phenotypes
- **AIT** (Asymmetric Intentionality Theory) — strategic asymmetries between L1 (corporate/algorithmic) and L3 (human) agents
- **GIMT** (Generalized Intentionality Mismatch Theorem) — six failure modes in human-institutional interaction
- **EGT** (Evolutionary Game Theory, multilevel selection) — ESS non-compliance, replicator dynamics
- **HBU** (Heteronomous Bayesian Updating) — how professionals learn norms by observing reactions, not outcomes
- **PSO** (Parasitic Spontaneous Order) — Dawkins-Hayek synthesis for extractive institutional equilibria
- **CLI** (Constitutional Lock-in Index) — validated quantitative measure of institutional rigidity
- **IHR** (Institutional Hysteresis Rate) — irreversible institutional deformation
- **IEI** (Institutional Evolvability Index) — design criterion for evolvable institutions

## Key Theoretical Instruments

### CLI (Constitutional Lock-in Index)

```
CLI = 0.30×P + 0.25×D + 0.20×O + 0.25×E
```
- P = Precedent density | D = Doctrinal elaboration | O = Professional organization | E = Enforcement infrastructure
- Validated: R²=0.74, AUC=0.97 (60 cases, 4 jurisdictions)
- Thresholds: <0.30 flexible | 0.30–0.50 reformable | 0.50–0.70 severe lock-in | >0.70 requires regime change

### Dennett-Nash Gap (AIT)

```
Δ_i = SA_i + ICS_i - MP_i + SIP_i
```
- Empirical estimates: Tech ToS 15–48% | Argentine decrees 40–62% | Sovereign debt ~40pp

### HBU (Heteronomous Bayesian Updating)

```
P_A(H|E) = [P(E|H)^w(A) × P(H)] / P(E)   where w(A) > 1
```
