## Research Program

The **Extended Phenotype Theory of Law (EPT)** applies Dawkins's (1982) extended phenotype theory to legal and institutional systems, combining:

- **EPT** — norms as cultural replicators, institutions as extended phenotypes
- **AIT** (Asymmetric Intentionality Theory) — strategic asymmetry between L1 (corporate/algorithmic) and L3 (human) agents; Dennett-Nash Gap
- **GIMT** (Generalized Intentionality Mismatch Theorem) — six failure modes in human-institutional interaction
- **HBU** (Heteronomous Bayesian Updating) — professionals learn norms by observing authoritative reactions, not outcomes
- **PSO** (Parasitic Spontaneous Order) — Dawkins-Hayek synthesis for extractive institutional equilibria
- **CLI** (Constitutional Lock-in Index) — quantitative measure of institutional rigidity (validated R²=0.74, AUC=0.97, 60 cases, 4 jurisdictions)
- **IHR** (Institutional Hysteresis Rate) — irreversible institutional deformation
- **IEI** (Institutional Evolvability Index) — design criterion for evolvable institutions

## Key Instruments

### CLI (Constitutional Lock-in Index)
```
CLI = 0.30×P + 0.25×D + 0.20×O + 0.25×E
```
Thresholds: <0.30 flexible | 0.30–0.50 reformable | 0.50–0.70 severe | >0.70 requires regime change

### Dennett-Nash Gap (AIT)
```
Δ_i = SA_i + ICS_i − MP_i + SIP_i
```
Empirical: Tech ToS 15–48% | Argentine decrees 40–62% | Sovereign debt ~40pp

### HBU (Heteronomous Bayesian Updating)
```
P_A(H|E) = [P(E|H)^w(A) × P(H)] / P(E)   where w(A) > 1
```
