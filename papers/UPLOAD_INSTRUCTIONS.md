# PDF Upload Instructions

This folder is for the PDF versions of SSRN papers.

## How to add your papers

1. Download each paper PDF from SSRN while your account is accessible
2. Rename each file according to the naming convention: `{SSRN_ID}-{Short-Title}.pdf`
3. Commit and push to this repository
